# Lista 1 – Análise Léxica

Este repositório contém a **resolução completa e comentada da Lista 1 da disciplina de Compiladores**, com foco em **Análise Léxica**, expressões regulares, autômatos finitos (NFA e DFA) e especificação de tokens.

O material foi organizado de forma modular, com uma pasta para cada questão, facilitando o estudo, a correção e a reutilização do conteúdo.

---

## Informações da disciplina

- **Universidade:** Universidade Federal do Ceará (UFC) – Campus Quixadá  
- **Disciplina:** Compiladores  
- **Professor:** Lucas Ismaily  

---

##  Conteúdo abordado

- Expressões regulares
- Linguagens regulares e não regulares
- Autômatos finitos não determinísticos (NFA)
- Autômatos finitos determinísticos (DFA)
- Conversão de ER → NFA → DFA
- Definição de alfabetos de linguagens
- Formas léxicas de identificadores e palavras-chave
- Tokens e análise léxica
- Exemplos aplicados em:
  - Pascal
  - C
  - Java
  - Python
  - Fortran

---

##  Organização do repositório

O repositório está organizado da seguinte forma:

```text
.
├── enunciado/
│   └── Lista_Analise_Lexica.pdf
│
├── questao-01/   # Expressões regulares
├── questao-02/   # Linguagens não regulares
├── questao-03/   # Linguagem reconhecida por autômatos
├── questao-04/   # Conversão de ER para NFA (com diagramas)
├── questao-05/   # Conversão de NFA para DFA
├── questao-06/   # Alfabetos das linguagens
├── questao-07/   # Formas léxicas (identificadores e palavras-chave)
└── questao-08/   # Tokens e análise léxica
```

## Cada diretório de questão contém:

- Um arquivo README.md com a descrição do problema;
- Um ou mais arquivos .md com a resolução detalhada;
- Uma pasta figuras/ (quando aplicável), contendo diagramas de NFAs e DFAs.
